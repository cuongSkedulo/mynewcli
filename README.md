# Usage
<!-- usage -->
```sh-session
$ npm install -g mynewcli 
$ mynewcli  COMMAND
running command...
$ mynewcli  (-v|--version|version)
mynewcli /0.0.0 darwin-x64 node-v14.16.0
$ mynewcli  --help [COMMAND]
USAGE
  $ mynewcli  COMMAND
...
```

## `mynewcli  help [COMMAND]`

display help for mynewcli

```
USAGE
  $ mynewcli  help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
